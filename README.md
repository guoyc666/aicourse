# AI 辅助的学习系统

这是一个利用 AI 技术辅助学习的系统，旨在帮助用户更高效地获取和理解知识。

## 功能特点

1. **知识资源管理模块**

   - 教师可上传多模态资源（文本、PPT、PDF、视频、音频、图片等）
   - 系统自动解析资源，提取元数据（标题、作者、关键字、课程章节）
   - 资源与知识图谱节点/关系自动关联

2. **知识图谱构建与管理**

   - 基于 Neo4j 等图数据库构建课程知识图谱
   - 包含概念节点、知识点关系、资源节点、学生学习记录节点
   - 支持知识图谱可视化展示

3. **课程垂直大模型构建**

   - 选用开源大模型（如 LLaMA2、Bloom、ChatGLM 等）
   - 基于知识库进行微调（LoRA、PEFT 等）
   - 部署课程专属垂直领域模型

4. **RAG 技术集成**

   - 学生提问时，先检索知识库（向量数据库/知识图谱）
   - 检索结果与输入结合增强后，交给微调模型生成答案

5. **个性化学习路径推荐**

   - 基于知识图谱和学生学习情况，动态生成个性化学习路径
   - 推荐相关学习资源（文章、视频、练习题）

6. **基于知识图谱的题库与评估**

   - 构建与知识点绑定的题库（选择题、填空题、编程题）
   - 学生答题后，系统根据答题情况更新知识点“掌握度”权重
   - 针对短板知识点，智能推荐个性化学习资源

7. **学习过程可视化**

   - 以知识图谱形式展示学生学习进度
   - 节点标注完成情况（已学、未学、掌握度百分比）
   - 展示学习质量指标（学习时长、答题正确率、完成度）

8. **多角色支持**

   - 教师：资源上传、题库维护、布置任务、查看学生学习情况
   - 学生：学习资源访问、答题、学习路径浏览、提问互动
   - 管理员：系统维护、用户管理、数据管理

9. **AI 助教**
   - 基于微调后的课程模型
   - 支持学生提问，结合学生画像与知识图谱，提供个性化解答和学习建议

## 技术架构

- **资源层**：多模态资源上传与解析，知识图谱/向量库管理
- **知识层**：Neo4j 知识图谱、题库、学生画像
- **模型层**：课程垂直大模型、RAG pipeline、AI 助教
- **应用层**：学习路径推荐、题库评估、学习进度可视化
- **用户层**：教师、学生、管理员
